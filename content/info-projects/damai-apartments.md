Este proyecto tiene el objetivo de llevar la gestión de los apartamentos de un edificio, desde el registro de los propietarios, hasta el control de los pagos de mantenimiento. 

## Skills
- NodeJS
- NestJS
- Vue 3
- NuxtJS
- TypeORM
- SQLite
- Docker
- Docker Compose
- PM2
